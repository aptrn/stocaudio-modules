# stocaudio-modules

Modules for [VCV Rack](https://github.com/VCVRack/Rack), open-source Eurorack-style virtual modular synthesizer.

## Plugins
![](stocaudio-modules.PNG)


## Building
Build dependencies loceally:

```bash
make dep
```
then build plugins using:

```bash
make
```
